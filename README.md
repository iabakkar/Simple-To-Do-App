# 📝 Todo List App

A simple Todo List application built using HTML, CSS, and JavaScript. This project allows users to add and delete tasks dynamically using DOM manipulation.

## 🚀 Features
✨ Add new todo items  
🗑️ Delete individual tasks  
⚡ Instant update without page reload  
🎯 Simple and clean UI  

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript (DOM Manipulation)  

## 🧠 How It Works
User types a task in the input field, clicks the Add button, and the task is added to the list dynamically. Each task has a Delete button that removes it from the list.
